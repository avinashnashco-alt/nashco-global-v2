# Cloudflare Worker for Nashco Global
#
# 1. Replace YOUR_ACCOUNT_ID in wrangler.toml with your Cloudflare account ID.
# 2. (Optional) Set route and zone_id for custom domain deployment.
# 3. To test locally: Install wrangler CLI and run `wrangler dev`
# 4. To deploy: Run `wrangler publish`
#
# worker.js is a simple example. Extend it as needed for API, proxy, or custom logic.
